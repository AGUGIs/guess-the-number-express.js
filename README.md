# 🎮 Угадай число — игра на HTML, CSS и JavaScript

Простая, но красивая браузерная игра «Угадай число от 1 до 100» с современным дизайном и плавными анимациями.  
Идеально подходит как мини-проект для портфолио или обучения веб-разработке!

---

## 🕹️ Как играть?

1. Компьютер загадывает случайное число от **1 до 100**.
2. У тебя есть **7 попыток**, чтобы угадать его.
3. Вводи число в поле и нажимай **«Проверить»** (или Enter).
4. Получай подсказки:  
   - 🔺 «Слишком мало!»  
   - 🔻 «Слишком много!»
5. Если угадаешь — получишь поздравление!  
   Если потратишь все попытки — узнаешь правильный ответ.

Нажми **«🔄 Начать заново»**, чтобы сыграть снова!
